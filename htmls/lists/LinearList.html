Config


During an animation no other function can be started until it's finished.

Animation can be stopped at any time using the pause button. The pause button will turn into a play button. Resume by pressing the play button.

While an animation is paused, click on the step button to continue step by step.

Animation speed ranges from 0 to 10, with 0 being manual mode and 10 being the fastest. While manual mode is active, animations don't start automatically. Instead only the first step gets executed immediately. When pressing the play button in manual mode, the current operation will proceed like in automatic mode with a speed value of 5.
