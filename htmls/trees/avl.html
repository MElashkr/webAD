Info

The AVL-tree is a sort of binary tree, which is balanced. As a binary tree every node can have up to two children. The left branch contains keys, which are less than the key of the node, the right branch stores the children with higher key-values. A tree is balanced, if the height of the children differs no more than one. An empty link has the height of 0.

To keep the tree balanced the AVL-tree extends the insert operation and the remove operation of a binary tree by rotations.
A rotation is performed, if the difference between the height of the left and the right branch is equal to two.

4 types of rotations are possible:
  1. left rotation
  2. right rotation
  3. left-right rotation
  4. right-left rotation
To visualize these rotations in this tool try the following combination of steps:
  1. empty tree, add certain value (1 2 3)
  2. empty tree, add certain value (3 2 1)
  3. empty tree, add certain value (3 1 2)
  4. empty tree, add certain value (1 3 2)

For more information regarding the AVL-tree visit: https://en.wikipedia.org/wiki/AVL_tree
